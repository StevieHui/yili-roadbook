# 伊犁环线自驾路书网站设计

## 目标

为 7 名约 22 岁旅客制作一套可分享、可在手机查看、可打印留存的伊犁七日自驾路书。行程为 2026 年 7 月 15 日抵达伊宁休整，7 月 16 日至 22 日完成伊宁往返环线，7 月 22 日晚或 23 日返航。

网站必须同时解决三个问题：快速理解全程、在路上按日执行、明确知道何时何地最适合拍照。

## 产品形态

采用单页故事地图，不引入后台和账号系统。页面由固定顶部导航串联以下锚点：

1. 首页封面
2. 方案总览
3. 交互路线图
4. 日历行程
5. 七日分段路书
6. 摄影机位
7. 必带清单
8. 关键提醒

桌面端强调电影感与地图联动，移动端强调快速查阅、单手操作和弱网降级。打印样式隐藏导航、动效和地图控件，保留行程、地址、清单与提醒。

## 视觉系统

### 视觉主题

主题名为“向天山深处”。整体不是常见的绿色旅游卡片模板，而是现代户外杂志与公路电影片头的结合：大幅景观、克制留白、强字号层级、路线编号和少量胶片注释。

### 色彩与字体

- 主色：天山墨绿 `#12352F`
- 湖水蓝：`#3D8D9E`
- 日落橙：`#E46F3A`
- 草甸浅绿：`#DCE7D5`
- 暖纸白：`#F5F1E8`
- 墨色：`#17201D`
- 中文标题使用有人文感的衬线字体栈，正文使用清晰的无衬线字体栈；字体加载失败时必须有系统字体降级。

### 版式与动效

- 首屏使用原创北疆横幅景观图，叠加路线主题、日期和一句短引导语。
- 总览数据采用横向信息带，不使用同质化圆角卡片墙。
- 每日路书交替使用大图、时间轴和驾驶信息，形成编辑式节奏。
- 动效只用于首屏轻微视差、路线绘制、区块淡入和导航状态；尊重 `prefers-reduced-motion`。
- 所有正文保持足够对比度，交互目标在移动端不小于 44px。

## 路线方案

全程约 1,200 km。里程与时间为高德规划参考值，山区、景区入口排队和临时管制不包含在内。

### 7 月 15 日：抵达伊宁

- 安排：取车、验车、采购饮水与路餐、入住休息。
- 验车重点：轮胎、备胎、刹车、玻璃水、随车工具、保险和车辆限高。
- 住宿：伊宁市区。

### Day 1 · 7 月 16 日：伊宁 → 赛里木湖

- 参考驾驶：153 km / 2 小时 22 分。
- 建议 09:00 前出发，中午进入景区，留足半日环湖和日落。
- 景观：高山湖泊、雪山、草甸、湿地。
- 出片点：月亮湾湖岸、克勒涌珠清澈水面、亲水滩礁石、松树头高处、日落公路弯道。
- 摄影提示：湖面顺光适合上午，日落前 60–90 分钟拍公路、人物和逆光草甸。
- 住宿：赛里木湖景区内或东门附近；优先选择可免费停车、夜间保暖条件明确的住宿。

### Day 2 · 7 月 17 日：赛里木湖 → 果子沟 → 霍城 → 伊宁

- 参考驾驶：约 238 km / 4 小时 18 分。
- 清晨补拍赛湖，之后经果子沟大桥前往霍城，傍晚返回伊宁。
- 景观：湖泊晨光、峡谷桥梁、薰衣草或河谷人文。
- 出片点：赛湖晨雾、果子沟大桥安全观景台、薰衣草园低机位、伊宁六星街蓝调时刻。
- 弹性规则：7 月中旬大田薰衣草可能已收割，出发前一天确认花况；花况不佳时改为漫心忘忧谷、惠远古城或直接增加伊宁人文拍摄。
- 住宿：伊宁市区。

### Day 3 · 7 月 18 日：伊宁 → 喀拉峻 → 特克斯

- 参考驾驶：伊宁至游客中心 133 km / 2 小时 37 分；景区至特克斯约 15 km。
- 07:30 出发，选择喀拉峻草原主线，不在一天内硬塞全部阔克苏峡谷项目。
- 景观：立体草原、雪岭、峡谷、牧群。
- 出片点：鲜花台层叠草甸、猎鹰台远景、人体草原侧光、牧道与马群。
- 摄影提示：广角拍层次，中长焦压缩草坡与雪山；避免踩入未开放草场。
- 住宿：特克斯八卦城。

### Day 4 · 7 月 19 日：特克斯 → 库尔德宁

- 参考驾驶：92 km / 1 小时 56 分。
- 上午从特克斯出发，中午前抵达，下午深度游览云杉森林。
- 景观：雪岭云杉、山谷溪流、森林光束。
- 出片点：十里画廊、云杉林栈道、喀班巴依峰方向、溪流慢门机位。
- 摄影提示：森林明暗反差大，人物穿浅色衣服；携带擦镜布和轻便雨具。
- 住宿：库尔德宁镇或景区周边。

### Day 5 · 7 月 20 日：库尔德宁 → 那拉提

- 参考驾驶：159 km / 3 小时 14 分。
- 上午离开库尔德宁，下午游那拉提空中草原主线，避免同时购买多条重复区间车线路。
- 景观：河谷、空中草原、雪山、毡房与牧场。
- 出片点：天界台、游牧人家附近草坡、雪莲谷方向、河谷草原日落。
- 摄影提示：下午侧光适合草坡层次；人物与马匹保持安全距离。
- 住宿：那拉提镇。

### Day 6 · 7 月 21 日：那拉提 → 独库北段 → 乔尔玛 → 唐布拉 → 尼勒克

- 强度：全程最重，预计 6–8 小时驾驶与停靠，必须早起。
- 07:30 前完成加油和早餐，按预约时段进入独库北段；19:00 前驶出受控路段。
- 景观：高山垭口、盘山公路、雪线、峡谷、百里画廊。
- 出片点：独库安全停车区的 S 弯、乔尔玛纪念地、喀什河谷、唐布拉百里画廊临河草地。
- 规则：仅在正式停车区停车；不在弯道、桥梁、落石区或道路中央拍摄。仙女湖徒步不塞入当天主计划。
- 住宿：尼勒克县城。

### Day 7 · 7 月 22 日：尼勒克 → 伊宁

- 参考驾驶：118 km / 1 小时 46 分。
- 上午沿喀什河谷返程，预留洗车、还车和航班机动时间。
- 景观：河谷、农田、水库、公路收束感。
- 出片点：合规观景点的河谷公路、伊犁河日落；若当天返航则取消日落安排。
- 住宿：仅 7 月 23 日返航时住伊宁市区。

## 地图与数据

### 地图行为

- 使用高德地图 JS API 2.0 显示真实底图。
- 默认展示全程线路，并以七种相近但可区分的颜色标示每日分段。
- 点击日期、日历卡片或每日路书时，地图聚焦对应分段并高亮当天 POI。
- 摄影点使用相机图标，住宿点使用床铺图标，风险点使用警示图标。
- 地图加载失败时显示可读的静态路线摘要，不影响正文和导航。

### 数据边界

- 行程内容、POI、时间、提醒和摄影建议集中存放在结构化数据文件，不散落在组件中。
- API Key 和 `securityJsCode` 从本地环境变量注入，不提交真实值。
- 生产部署前必须轮换已在聊天中暴露的两项凭据，并配置域名限制。
- 景区票价、开放时间和独库路况标记为“出发前复核”，避免把易变信息写成永久事实。

## 关键提醒

页面必须突出以下行动项：

- 赛里木湖、喀拉峻自驾名额和热门住宿提前预约。
- 2026 年独库北段按时段预约，7 座及以下车辆方可按当前规则预约；两辆车分别预约。
- 每车至少两名可轮换驾驶员；每天出发前确认油量、胎压和路况。
- 身份证、驾驶证、行驶证、租车合同和保险资料随车携带。
- 山区温差、紫外线和天气变化大，准备抓绒/薄羽绒、防晒、雨具和饮水。
- 不在草场碾压、不追逐牲畜、不飞越禁飞区；无人机起飞前查询现场规定。
- 下载高德离线地图，并把住宿、景区入口和集合点保存到两台以上手机。

## 清单交互

必带清单按“证件车辆、衣物防护、摄影电子、食品药品、团队公共物资”分组。勾选状态保存在浏览器本地；提供一键重置。页面不收集个人数据。

## 技术架构

- Vite + React + TypeScript。
- 组件按封面、总览、地图、日历、日程、摄影、清单、提醒拆分。
- Vitest + Testing Library 验证路线数据、导航、日期切换、地图降级和清单持久化。
- CSS 使用项目内设计令牌与原生响应式布局，不引入笨重 UI 组件库。
- 图片采用 WebP/AVIF 优先、JPEG 回退，并设置尺寸避免布局偏移。

## 验收标准

- 桌面 1440px、平板 768px、手机 390px 均无横向溢出。
- 首屏在地图尚未加载时仍完整可见。
- 所有导航锚点可到达，当前区块状态可感知。
- 点击任意一天会同步更新地图分段与日程详情。
- 高德脚本失败时显示降级内容且无未处理异常。
- 清单刷新后保持勾选状态，重置操作生效。
- 键盘可访问主要导航与交互控件。
- 打印预览包含七日行程、住宿、清单和关键提醒。
- 测试、类型检查和生产构建全部通过，浏览器无控制台错误。
