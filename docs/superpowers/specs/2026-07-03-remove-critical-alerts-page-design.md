# 删除独立关键提醒页面设计

## 目标

删除顶部导航中的独立“关键提醒”页面，减少低价值入口和重复信息。

## 范围

- 从 `SiteHeader` 删除“关键提醒”导航项。
- 从 `App` 删除 `alerts` 视图类型、组件导入和渲染分支。
- 删除不再使用的 `CriticalAlerts` 组件。
- 删除仅供独立提醒页面使用的 `.alerts-*` 样式，并清理与其他选择器合并书写的提醒页面片段。
- 更新导航测试，断言“关键提醒”不再作为导航入口出现。

## 保留内容

保留 `RouteExplorer` 中 DAY6 的“关键提醒”和独库预约说明；它属于当天路线的上下文信息，不在本次删除范围内。

## 验证

运行全量测试、TypeScript 类型检查和生产构建，确认不存在失效视图、残留导入或样式引用。
